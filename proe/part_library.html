ProE零件库的建立

第一步:设置零件库目录

  • 在E盘下新建一个文件夹目录,做为零件库目录,命名为:part_library

第二步:设计环境变量

  • 把刚才的零件库目录加入电脑的环境变量

第三步:把常用标准零件分类放置

第四步:建立mnu文件

  • 在零件库的根目录,新建一个后缀为mnu的文件

第五步:填写零件信息

  • 总目录格式如下: 
    第一行为零件库标题,可自定义
#            //零件库名称的分隔符
 #           // 分隔符
 /螺丝类     //目录
 常规螺丝    //说明
 #           //分隔符
 /弹簧类     //目录
 常用弹簧    //说明
 #
 /螺母类
 常用型
 #
  • 子目录格式如下(如十字螺丝目录): 
    第一行标题:常用螺丝子库---十字螺丝
#
#
1-123.prt
ST2.9*16,锁大压线板用的螺丝
#
1-124.prt
ST2.9*14,锁小压线板用的螺丝
#
  • 其所有目录都需要建立mnu文件,且mnu文件名与上级文件名相同,以便proe识别读取零件

第六步:建立catlog(分类)文件

  • Mnu文件是告诉proe在读当前目录时该怎么样处理目录下的子项。而catalog文件则是告诉proe所有的零件库的零件该如何去寻找，这样才能在不同的装配中任何时候打开都能找到所需要的常用零件而不至于在关掉proe重开后就发生零件的丢失现象。
  • 运行CMD,打开DOC窗口
  • 进到零件库目录,运行pro_build_library_ctg.bat即可创建零件库的catalog文件

第七步:设置配置文件

  • 设置以下配置文件, 
        !设置配置零件库路径目录
    pro_library_dir e:\proe_library
    !设置catalog目录
    pro_catalog_dir e:\proe_library
    !设置不修改不保存新版本选项
    save_objects changed
