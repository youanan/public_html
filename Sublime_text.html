Sublime text 安装

  • Sublime text 官网:https://www.sublimetext.com/
  • Sublime text3 官网下载地址:https://www.sublimetext.com/3
  • Sublime text 激活码:https://gist.github.com/axhello/b97306fb3d1b297cf4e2b082214d619d
  • Sublime text插件 :https://packagecontrol.io/installation#st3
    1. 按Ctrl+ ` ,打开控制台;
    2. 输入: import urllib.request,os,hashlib; h = '6f4c264a24d933ce70df5dedcf1dcaee' + 'ebe013ee18cced0ef93d5f746d80ef60'; pf = 'Package Control.sublime-package'; ipp = sublime.installed_packages_path(); urllib.request.install_opener( urllib.request.build_opener( urllib.request.ProxyHandler()) ); by = urllib.request.urlopen( 'http://packagecontrol.io/' + pf.replace(' ', '%20')).read(); dh = hashlib.sha256(by).hexdigest(); print('Error validating download (got %s instead of %s), please try manual install' % (dh, h)) if dh != h else open(os.path.join( ipp, pf), 'wb' ).write(by) Enter
    3. 自动下载安装
  • 安装中文插件:
    1. 按下Ctrl + shift + p, 打开命令窗口:输入 Install Package , Enter;
    2. 输入Chinese,选择 ChineseLocalization,Enter;
    3. 切换中英文: 帮助 > Language ...

Sublime text 插件：

  • *Emmet*:快速生成HTML代码段的插件
  • ConvertToUTF8 :支持 GBK, BIG5, EUC-KR, EUC-JP, Shift_JIS 等编码的插件
  • Bracket Highlighter:用于匹配括号，引号和html标签。对于很长的代码很有用。安装好之后，不需要设置插件会自动生效
  • JsFormat:自动格式化js代码
  • SideBarEnhancements: 这个插件改进了侧边栏,增加了很多功能
  • Anaconda: python插件
  • iew_in_browser 在浏览器中打开文件
