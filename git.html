git本地设置

git安装完成后的操作

  1. 设置config信息
    1. git config --global user.name "yourname" //设置用户名称
    2. git config --global user.email "youremail"` //设置用户邮箱
    3. git config --list //查看config配置的信息
  2. 设置SSH keys文件
    1. 首先检测文件是否存在，如存则跳过此步骤
      • cd ~/.ssh //进入ssh文件夹
      • ls //查看文件夹内有哪些文件，关键是找.rsa文件
    2. 将原来的SSH keys备份并删除
      • mkdir key_backup 新建一个key_backup的文件夹s
      • cp id_rsa* key_backup //复制id_rsa开头的文件到key_backup文件夹内
      • rm id_rsa* //删除现有id_rsa开头的文件
    3. 创建一个新的SSH key
      • ssh-keygen -t rsa -C "yourgithub's email"
      • //回车，默认
      • //回车，使用默认路径
      • //输入key密码,不输入回车则不需要密码
    4. 将is_rsa.pub的内容复制到github>settings>SSH keys中
    5. 测试是否可以与github正确连接
      • ssh -T git@github.com //出现Hi......开头的英文，就是连接正常
